一、音乐模式支持的操作：
	①、暂停/播放。		//key0短按
	②、下一曲。	 		//key1短按
	③、上一曲。   		//key2短按
	④、下一个文件夹。//key1长按，循环文件夹 dir_song -> dir_eng -> dir_poetry -> dir_story -> dir_midi -> dir_song
	⑤、a背景音。			//key2长按

二、io key：
	KEY0 -> PA7
	KEY1 -> PA8
	KEY2 -> PA9
